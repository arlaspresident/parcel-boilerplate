Arbetsprocess

Syfte

Projektet syftar till att skapa en boilerplate för webbutveckling. Den ska innehålla en grundläggande struktur för HTML, CSS, JavaScript och bilder samt automatisering med hjälp av Parcel för att optimera filer och effektivisera arbetsflödet.

Arbetssteg

  1. Skapa mappstruktur: En mappstruktur skapades med mappar för css, js och images.
  2. HTML sidor: Grundläggande HTML sidor skapades: index.html, images.html, och process.html.
  3. CSS: En fil för styling (style.css) skapades och grundläggande styling lades till för layout och galleriet.
  4. JavaScript: En JavaScript-fil (script.js) lades till med grundläggande funktioner, inklusive DOMContentLoaded-händelsen.
  5. Parcel: Parcel installerades och konfigurerades för att hantera utvecklingsservern och bygga projektet.
  6. Versionhantering: Git initierades och projektet lades till i ett GitHub repo med regelbundna commits.

Använda verktyg

Följande verktyg användes:

  • Node.js och Parcel: För att automatisera utvecklingsflödet och optimera filer.
  • Git och GitHub: För versionshantering och publicering av projektet.
  • HTML, CSS och JavaScript: För att bygga och strukturera webbplatsen.

Publiceringsprocess

Webbplatsen kommer att publiceras via Netlify. Publiceringen sker automatiskt när uppdateringar pushas till GitHub.
